Resume/ CV / Vcard Html Template

  • latest update: 07/31/2020

Description

IT'S ME is a modern and premium v-card, cv, resume template with a beautiful design manner, it allows you to introduce yourself with the most important informations to your futur clients or employers with a unique and a professional way.

The template is responsive that’s allows it to view it in the mobile devices also.

All the html ,css, javascript code are well organized and commented to make any change easy to do without any problems.

Features

  • Premium design
  • Retina ready
  • Parallax animation
  • Built using bootstrap
  • Project single page
  • Blog pages
  • Filterabe portfolio

 

Theme installation

When you download the main download file you will  find several folder  :

  •       HTML folder this is the folder that contain all the files of the template
  •       LICENSES folder this folder contain the licenses of the elements used.
  •       DOCUMENTATION : this folder contain the documentation of the theme please read the documentation carefully for further information

File Structure

In this section a guide to understanding how IT'S ME is structured.

CSS subfolder 

This folder contain all the css files that are required by the template those files are :

  1. main.css
  2. bootstrap.css
  3. venobox.css
  4. font.css
  5. owl carousel subfolder that style the carousel testimonials
  6. animate.css

JS subfolder

This folder contain all the js files required by the template those files are :

  1. jquery
  2. bootstrap
  3. venobox
  4. owl carousel
  5. smooth-scroll.js
  6. parallax
  7. placeholders
  8. easing
  9. script

ASSETS subfolder

This subfolder contain other subfolders :  the img subfolder and the svg subfolder, evenif those are not included in the main download files and replaced with placeholders, but you can replace theme easly with your own assets.

  • Images used in html pages should be found in /assets/img.
  • Svg used in html pages should be found in /assets/svg.

 

WEBFONTS subfolder

This folder contain all the webfonts from fontawsome used in this template, you can use a wide range of icons that suits your need and the . For more info on how to setup and use fontawsome please visit this link below :

https://fontawesome.com/how-to-use/on-the-web/setup/getting-started?using=web-fonts-with-css

Back To Top Button

The back to the top button is controlled by the scrollTo plugin to give the user the ability to reach the top of the browser if the content is too long. Usually, it will be inserted at the end of the page before the end of the body tag.

Custom Pointer setup

IT'S ME comes with a cool custom effect pointer to add some animation when navigating the template, this effect can be removed easily by opening the script.js file located in the js folder you will have to find the section below and comment the code so the effect will be disabled. 

        // Custom cursor

        $(document).ready(function(){
            var clientX=-200;
            var clientY=-200;
            var bottomCursor=document.querySelector('.circle-cursor-bottom');
            var topCursor=document.querySelector('.circle-cursor-top');


            function displayCursor(e){
                document.addEventListener('mousemove',e=>{
                clientX=e.clientX;
                clientY=e.clientY
            });


            function animCursor(){
                    topCursor.style.transform=`translate(${clientX}px, ${clientY}px)`;
                    bottomCursor.style.transform=`translate(${clientX}px, ${clientY}px)`;
            requestAnimationFrame(animCursor)};
            requestAnimationFrame(animCursor)};
            displayCursor();
        
        });
